## מה זה עושה

כפתור אחד שמוריד את כל הקורס שלך — קבצים, תיקיות, מטלות, סילבוס, הקלטות וקישורים — לתוך ZIP מסודר אחד עם תיקיות, ויש גם קובץ דדליינים שאפשר לייבא ליומן.

עובד על: `https://moodlearn.ariel.ac.il/*`

## פיצ'רים עיקריים

- 🗂️ **סריקה לפי קטעים** עם בורר שמאפשר לבחור בדיוק מה להוריד
- 📦 **קבצים, תיקיות, מטלות, הגשות** — הכל מסודר בתיקיות נפרדות בתוך ה-ZIP
- 📅 **דדליינים → קובץ ICS** לייבוא ל-Google Calendar / Outlook / Apple Calendar
- 🎥 **זיהוי הקלטות** (Zoom, Panopto, Kaltura, YouTube, Webex, Teams ועוד) ושמירתן כקישורים בקובץ נפרד
- 📄 **סילבוס וקישורים חיצוניים** — מנסה להוריד אוטומטית אם זה PDF/Office, אחרת שומר כקישור
- 🆕 **מצב Diff** — בסריקה השנייה ואילך, יודע מה כבר ירד ומציע להוריד רק מה שחדש
- 📚 **הורדת מספר קורסים** — בדף "הקורסים שלי" אפשר לבחור כמה קורסים ולקבל ZIP נפרד לכל אחד
- 🖱️ **קליק ימני** — "הורד עם Moodle Hoarder" על כל קישור במודל
- 🎬 **Zoom Cloud Recordings** — בעמוד הקלטות Zoom שנפתח דרך מודל, לחיצה על "סרוק" מפיקה קובץ טקסט עם כל הקישורים, התאריכים והשמות
- 🌙 **Dark mode** אוטומטי, RTL מלא, נוטיפיקציות מערכת

לרשימה מלאה: [FEATURES.md](FEATURES.md)

## כלי תמלול מקומי ניסיוני

נוסף MVP נפרד ב-[`transcriber/`](transcriber/) לתמלול הקלטות קורסים בעברית עם faster-whisper. הכלי כולל GUI קטן ו-CLI, תומך בבחירת מודל, ומייצא `txt`, `srt`, `vtt` ו-`json` עם timestamps.

## התקנה

התוסף לא בחנות הרשמית של Chrome. ההתקנה ידנית (לוקחת 30 שניות):

1. הורד את הפרויקט: [Download ZIP](https://github.com/eitanav/moodle-hoarder/archive/refs/heads/main.zip) או `git clone`
2. חלץ את ה-ZIP למקום קבוע במחשב (לא בתיקיית Downloads — אם תמחק את התיקייה התוסף יישבר)
3. פתח `chrome://extensions/` בכרום
4. הפעל **Developer mode** (מתג ימני עליון)
5. לחץ **Load unpacked** ובחר את התיקייה
6. נעץ את האייקון של Moodle Hoarder לסרגל הכלים (אייקון הפאזל)

## איך משתמשים

### הורדת קורס בודד

1. היכנס לדף קורס כלשהו במודל (`course/view.php?id=...`)
2. לחץ על האייקון של התוסף
3. לחץ **"סרוק"**
4. סמן/בטל סימון של מה שאתה רוצה (יש דיפולטים חכמים — פורומים וכו' לא מסומנים)
5. לחץ **"הורד ל-ZIP"**

### הורדת כל הקורסים שלך

1. עבור ל-`https://moodlearn.ariel.ac.il/my/courses.php`
2. פתח את התוסף → "סרוק" → סמן את הקורסים → "הורד את כל הקורסים שנבחרו"
3. כל קורס יורד כ-ZIP נפרד

### ייצוא הקלטות Zoom של קורס

1. בעמוד הקורס במודל, לחץ על פעילות Zoom (פעילות LTI)
2. דף הקלטות Zoom נטען
3. פתח את התוסף → "סרוק"
4. יורד `zoom-recordings_<תאריך>.txt` עם רשימת הקישורים והמטא-דאטה

> טיפ: בהורדת MP4, אם הנגן המובנה של Windows מציג `unsupported encoding settings`, פתח את הקובץ עם VLC. זה בדרך כלל אומר שהנגן המובנה לא תומך בהגדרות הקידוד של Zoom, לא שההורדה פגומה.

### דדליינים ביומן

בתוך כל ZIP יש קובץ `deadlines.ics`. דאבל-קליק עליו פותח את היומן של מערכת ההפעלה ומציע לייבא — כל הדדליינים של הקורס נכנסים עם תזכורת יום מראש.

## מבנה ה-ZIP

```
שם הקורס.zip
├── *.pdf, *.docx, ...          ← קבצים בודדים (resource)
├── folders/
│   └── <שם תיקייה>.zip         ← תיקיות שלמות
├── assignments/
│   └── <שם מטלה>/              ← חומרי המטלה
├── submissions/
│   └── <שם מטלה>/              ← מה שאתה הגשת
├── pages/
│   └── <שם דף>/                ← דפי Moodle Page + קבצים מוטמעים
├── urls/                       ← קבצים שירדו מקישורים חיצוניים (סילבוס וכו')
├── links.txt                   ← קישורים שאינם קבצים
├── recordings.txt              ← הקלטות שיעורים (Zoom, Panopto וכו')
├── deadlines.ics               ← דדליינים של מטלות לייבוא ליומן
└── info.txt                    ← סיכום ההורדה
```

## פרטיות

- התוסף **לא שולח שום מידע אישי, תוכן קורס או דאטה לשום שרת**.
- כל עיבוד הקבצים רץ מקומית בדפדפן שלך.
- ה-cookies של המודל שלך משמשים לאימות בדיוק כמו שהדפדפן עושה רגיל.
- מה ש"זוכר" התוסף (רשימת קבצים שירדו, היסטוריה, הגדרות) נשמר ב-`chrome.storage.local` רק על המחשב שלך.
- **הבקשה היחידה לאינטרנט** (מעבר למודל/Zoom עצמם) היא בדיקת עדכונים אופציונלית: GET לקובץ `manifest.json` הציבורי ב-GitHub כדי לבדוק אם יצאה גרסה חדשה. לא נשלח בה שום מידע, וניתן לכבות אותה בהגדרות. ראה [`PRIVACY.md`](PRIVACY.md).

## תרומה ופיתוח

הקוד כולו vanilla JS, ללא תלויות חיצוניות, ללא בנייה. כדי לפתח:

```bash
git clone https://github.com/eitanav/moodle-hoarder.git
cd moodle-hoarder
# Load unpacked מ-chrome://extensions/, ערוך קבצים, רענן (R) בכרטיס התוסף
```

מבנה הקוד:
- `manifest.json` — הגדרות התוסף
- `popup.html` / `popup.js` — ממשק המשתמש והלוגיקה הראשית
- `zip.js` — ZIP writer מינימלי (store-only, ללא דחיסה)
- `background.js` — service worker לקליק-ימני
- `icons/` — לוגו ב-4 גדלים
- `generate-icons.ps1` — סקריפט PowerShell לבניית הלוגו מחדש
