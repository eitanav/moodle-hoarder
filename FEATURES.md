# Moodle Hoarder

תוסף Chrome להורדה אוטומטית של כל החומר מקורסים במודל של אוניברסיטת אריאל.

גרסה נוכחית: **1.2.0**

---

## פיצ'רים קיימים

### זיהוי עמוד אוטומטי
- מזהה עמוד **קורס בודד** (`/course/view.php?id=...`) → מציג בורר פריטים.
- מזהה עמוד **הקורסים שלי** (`/my/courses.php`) → מציג בורר קורסים להורדה מרובה.
- אם לא בעמוד תומך — מסביר מה צריך.

### סריקה לפי קטעים
- הפריטים מקובצים לפי **הקטעים של הקורס** (Topics) כפי שהם בנויים במודל.
- לכל קטע: שם, ספירת פריטים, צ'קבוקס "אב" שמסמן/מבטל את הכל, קליק על הכותרת מקפל/פותח.
- הצ'קבוקס הראשי יודע מצב Indeterminate (חצי מסומן).
- חיפוש חי בתוך הפריטים.
- כפתורי בחירה: **הכל / ניקוי / איפוס לדיפולטים**.

### ברירות מחדל חכמות
- תמיד לא מסומן: `forum`, `chat`, `feedback`, `choice`, `wiki`, `glossary` — פעילויות שלא נוצרות לקבצים.
- בקטע הראשון (מבוא) גם `url` לא מסומן — כי לרוב מדובר בקישורים פחות חשובים.
- כל השאר מסומן.

### סוגי פעילויות נתמכים

| סוג | מה קורה | יעד בתוך ה-ZIP |
|-----|---------|---------------|
| `resource` | מוריד את הקובץ עצמו, עם שם המקור מ-Content-Disposition | שורש ה-ZIP |
| `folder` | משתמש ב-`download_folder.php` להורדה רקורסיבית של כל המבנה הפנימי. אם חסום — סקרייפינג ידני | `folders/<שם>.zip` |
| `assign` | קבצי המטלה + הגשות שלך + שליפת דדליין | `assignments/<שם>/` + `submissions/<שם>/` |
| `url` | חכם — מנסה להוריד אם זה קובץ, אחרת שומר כקישור | `urls/<שם>` או `links.txt` |
| `page` | מוריד את הקבצים המוטמעים + שומר HTML מעוצב להדפסה | `pages/<שם>/` |
| `book` | שומר HTML של ה-book | `books/<שם>.html` |
| הקלטות | זיהוי דומיינים (Zoom/Panopto/YouTube וכו') | `recordings.txt` |

### זיהוי סילבוס וקבצים חיצוניים
- ל-`mod/url` שמצביע ל-PDF/Office/Zip: ניסיון אמיתי לפתוח, בדיקת `Content-Disposition: attachment` ו-`Content-Type` כדי להחליט אם להוריד או לשמור כקישור.
- חוקי דומיין: רק `*.ariel.ac.il` נלקח בחשבון להורדה — דומיינים חיצוניים נשמרים כקישור ללא ניסיון רשת (מונע שגיאות CORS).

### הפרדת קישורים
- **`recordings.txt`** — הקלטות שיעורים בלבד (Zoom, Panopto, Kaltura, Mediasite, YouTube, Vimeo, GoTo, Webex, Teams, Meet, Jitsi, BigBlueButton, Twitch, ועוד).
- **`links.txt`** — שאר הקישורים החיצוניים (אתרים, מסמכים, פלטפורמות שיתוף).

### מצב Diff
- אחרי הורדה ראשונה, התוסף שומר את רשימת הפריטים שירדו ב-`chrome.storage.local` עם תאריך.
- בסריקה הבאה: באנר עליון "מצאתי הורדה קודמת מ-X, K חדשים".
- כל פריט חדש מקבל chip צהוב **"חדש"**.
- כפתורים מהירים: **"בחר רק חדשים"** / **"הכל"**.
- אפשר תמיד לבחור ידנית לפי קטע/פריט.

### הורדת מספר קורסים
- כשנמצאים בעמוד "הקורסים שלי" — רשימה מלאה של כל הקורסים עם צ'קבוקסים + חיפוש.
- מסמנים כמה קורסים, ה-תוסף מוריד **ZIP נפרד לכל קורס**.
- אופציה גלובלית: "הורד רק חדשים בכל קורס" — מחיל את ה-diff על כל קורס בנפרד.
- בכל ZIP מופעלים אוטומטית הדיפולטים החכמים (ללא בחירה ידנית פר קורס — מהירות מעל גמישות).

### לוח שנה (ICS)
- שולף דדליין מכל מטלה (`mod/assign`) — תומך בעברית, אנגלית, פורמטים DD/MM/YYYY ו-YYYY-MM-DD.
- בונה `deadlines.ics` בתוך ה-ZIP עם תזכורת יום אחד מראש.
- אפשר לייבא ישירות ל-Google Calendar / Outlook / Apple Calendar (פתיחה כפולה של הקובץ).

### תפריט קליק ימני
- קליק ימני על כל קישור במודל → **"הורד עם Moodle Hoarder"**.
- ממיר אוטומטית `mod/resource/view.php` ל-redirect ישיר לקובץ.
- ממיר `mod/folder/view.php` ל-`download_folder.php` להורדת ZIP של התיקייה.

### עיצוב וחוויית משתמש
- לוגו ייעודי ב-4 גדלים (16/32/48/128 px) עם גרדיאנט כתום-ורוד.
- Header צבעוני עם הלוגו.
- **Dark mode** אוטומטי לפי הגדרות מערכת ההפעלה.
- RTL מלא עם פונט Segoe UI.
- Progress bar בזמן אמת.
- לוג בזמן אמת עם צבעי הצלחה/שגיאה.
- ספירת נבחרים, ספירת תוצאות חיפוש.
- נוטיפיקציה במערכת ההפעלה כשהכל מסתיים.

### מטא-דאטה
- `info.txt` בכל ZIP: שם קורס, כתובת, תאריך, ספירת פריטים, רשימת מה שירד, רשימת שגיאות.
- BOM בקבצי הטקסט כדי שעברית תיראה תקין ב-Notepad של Windows.

### יציבות
- ZIP store-only — בלי תלות חיצונית, בלי כפילות זיכרון.
- שמות קבצים מנוקים (תווים בלתי-חוקיים → `_`, חיתוך ל-150 תווים).
- נתיבים כפולים מקבלים סיומת " (2)", " (3)" וכו'.
- כל הורדה כושלת נרשמת ב-`info.txt` בלי לעצור את שאר ההורדה.

## רעיונות לעתיד

### Phase 2 — תזכיר שלך
1. **ייצוא הקלטות מ-Zoom** — כניסה ל-`zoom.us/recording`, סריקת כל הקליפים של המשתמש, שמירת רשימה ל-`zoom-recordings.txt` (אופציונלית עם תאריכים, אורך, ושם הקורס המתאים).

### שיפורי שימוש
2. **המרת `page`/`book` ל-PDF אמיתי** — שילוב של `pdfMake` או `html2pdf.js` עם תמיכת RTL. כיום נשמר HTML ניתן להדפסה ידנית.
3. **קליק ימני מתקדם** — תפריט מתפצל: "הורד לתוך ZIP חדש" / "הוסף לתור הורדות". תור שאפשר לסקור ולשלוח לזיפ בסוף.
4. **שכפול מבנה הקטעים בתוך ה-ZIP** — כיום הכל בשורש; אפשרות לסדר כ-`section-01-מבוא/`, `section-02-שיעור-1/` וכו'.
5. **אופציה "Save As" לבחירת תיקייה** — היום נופל ל-Downloads. הוסף checkbox "פתח שמירה בשם".

### מודיעין על הקורס
6. **דשבורד היסטוריה** — מסך נפרד בתוסף שמראה את כל הקורסים שהורדת, מתי, כמה פריטים, וקישור מהיר להורדה חוזרת.
7. **השוואת גרסאות** — אם קובץ עם אותו שם הופיע שוב והוא שונה בגודל, סמן "עודכן" במקום "חדש" וצרף לתיקייה `updates/`.
8. **גרף פעילות בקורס** — כמה פריטים פר קטע, כמה דדליינים, כמה הקלטות. דשבורד קצר לפני ההורדה.
9. **שליפת ציונים** — סריקת `grader/report` ויצוא ל-CSV של כל הציונים שלך בקורס.

### חיפוש ותוכן
10. **חיפוש בתוך הקבצים** — אינדוקס מקומי של PDFים שירדו (pdf.js), חיפוש טקסט מלא בכל הקורסים שהורדת. בלי לעלות לענן.
11. **תקציר AI לכל קובץ** — שילוב Claude/OpenAI API עם מפתח של המשתמש. סיכום של כל מצגת/PDF בעת ההורדה.
12. **חילוץ טקסט מהקלטות** — אם להקלטות Panopto/Zoom יש כתוביות, הורדת ה-VTT/SRT.

### זרימה ויעילות
13. **תזמון אוטומטי** — הורדה אוטומטית של עדכונים פעם ביום/שבוע ברקע. נוטיפיקציה כשמשהו חדש.
14. **התראה לדדליין מתקרב** — בלי ICS, התראה ישירה מהתוסף יומיים לפני כל הגשה.
15. **הורדה במקביל** — כיום סדרתית; לעשות במקביל (4-6 בו-זמנית) כדי להאיץ קורסים גדולים.
16. **חידוש הורדה כושלת** — אם הזיפ נכשל באמצע, לזכור מה ירד ולהמשיך משם.

### מודולים נוספים במודל
17. **`quiz`** — שמירת ניסיונות שלך + שאלות + תשובות נכונות (אם זמינות לאחר ההגשה).
18. **`forum`** — כן, ברירת מחדל off, אבל למי שרוצה: הורדת כל הדיונים ל-HTML/Markdown.
19. **`lesson`** — שחזור כל המסכים של ה-lesson.
20. **`h5p`** — תוכן אינטראקטיבי. הורדת קובץ ה-H5P המקורי אם זמין.
21. **`scorm`** — שחרור חבילות SCORM (אינטראקטיבי).
22. **`workshop`** — סקירות עמיתים והגשות.

### בינלאומיות והרחבה
23. **תמיכה במוסדות נוספים** — תבנית להוספת מודלים אחרים (אונ' תל-אביב, חיפה, BGU, האו"פ). מבוסס Moodle ברובם → אותה לוגיקה.
24. **שפת ממשק** — כיום עברית; הוספת אנגלית/רוסית/ערבית.
25. **כפתור בתוך עמוד הקורס** — במקום לפתוח את הפופאפ, להזריק כפתור צף ישירות לעמוד המודל.

### עוצמה למפתחים
26. **ייצוא JSON של הקורס** — מטא-דאטה מלאה (פריטים, סוגים, IDs, תאריכים, נתיבים) ל-`course.json` בתוך ה-ZIP, נוח לסקריפטים.
27. **CLI / Headless mode** — חבילת Node שעושה את אותו דבר ברקע, בלי דפדפן. שימושי לגיבוי תקופתי בשרת.
28. **WebDAV / Drive sync** — סיים הורדה → העלאת ה-ZIP אוטומטית לתיקיית Drive/Dropbox/WebDAV.
29. **ספריית פלאגינים** — `mod/` handlers ניתנים להוספה. מישהו יכול להרחיב את התוסף בלי לגעת בליבה.

### פרטיות ואבטחה
30. **הצפנת ZIP** — סיסמה אופציונלית לזיפ (מצריך דחיסה אמיתית; כיום store-only).
31. **גיבוי מרובה-מכשירים** — סנכרון רשימת ה-"seen" דרך `chrome.storage.sync` כדי שמצב ה-diff יעבוד מכל מחשב.
32. **מחיקת היסטוריה** — כפתור "מחק את כל מה שהתוסף זוכר" עם דיאלוג אישור.

### עיצוב מתקדם
33. **תצוגה מקדימה לקבצים** — לפני הורדה, שורת hover עם גודל, סוג, מתי הועלה.
34. **גרירה לסידור** — אפשרות לשנות סדר/קיבוץ ידני בזיפ.
35. **שמות תיקיות מותאמים** — תבניות כמו `{course-id}_{name}_{date}.zip`.
