# Moodle Hoarder

תוסף Chrome להורדה אוטומטית של כל החומר מקורסים במודל של אוניברסיטת אריאל.

גרסה נוכחית: **1.8.0**

---

## פיצ'רים קיימים

### זיהוי עמוד אוטומטי
- **קורס בודד** (`/course/view.php?id=...`) → בורר פריטים.
- **הקורסים שלי** (`/my/courses.php`) → בורר קורסים להורדה מרובה.
- **דף הקלטות Zoom** (`applications.zoom.us/lti/rich`) → בורר הקלטות + פענוח URLs.

### דף הגדרות נפרד (v1.8.0)
- אייקון ⚙ בפופאפ פותח דף הגדרות מלא בטאב חדש.
- כל הטוגלים והאופציות שם — הפופאפ נשאר נקי לעבודה.

### מבנה ZIP מותאם (v1.6.0+v1.8.0)
- שלוש פריסות לבחירה בהגדרות:
  - **לפי קטעים** — תיקייה לכל קטע ("01 - מבוא", "02 - שיעור 1"...).
  - **הכל ביחד** — תיקייה שטוחה אחת עם כל הקבצים ("00 - כל הקבצים").
  - **שניהם** (ברירת מחדל) — ZIP מכיל את שני המבנים, גדל פי 2.
- מטלות מקבלות תיקייה משלהן, ההגשות שלך תחת `_הגשות שלי/`.

### שמות עברית תקינים (v1.7.1)
- ZIP נכתב עם Info-ZIP Unicode Path Extra Field — Windows מציג עברית נכון.
- תיקון אוטומטי של mojibake בשמות שמגיעים מכותרות `Content-Disposition` של Moodle.

### בחירה לפני הורדה
- חלוקה לפי קטעים עם צ'קבוקס "אב" לכל קטע.
- חיפוש חי, "סמן הכל / ניקוי / איפוס".
- **דיפולטים חכמים**: פורומים, צ'אטים, ויקי לא מסומנים אוטומטית.
- ניתן להגדיר בהגדרות אילו סוגי פעילויות מסומנים כברירת מחדל.

### חידוש הורדה כושלת (v1.8.0)
- אם הפופאפ נסגר באמצע הורדה — הקבצים שכבר ירדו נשמרים ב-chrome.storage.
- בסריקה הבאה של אותו קורס מופיע באנר "נמצאה הורדה לא-גמורה" וההורדה ממשיכה משם בלי להוריד שוב את מה שכבר יש.

### Diff מתקדם (v1.8.0)
- תווית **"חדש"** — פריט שלא היה בהורדה הקודמת.
- תווית **"לא בדיפולט"** — פריט שלא היה כי הוא מסוג שלא יורד בדיפולט (פורום וכו') — לא באמת חדש.
- כפתור מהיר "בחר רק חדשים" בבאנר.

### הורדה מקבילית (v1.7.0)
- עד 5 פריטים בו-זמנית. קורסים גדולים יורדים פי 3-5 מהר יותר.
- ZIP ולוג נשארים בסדר המקורי.

### דדליינים → קובץ ICS
- כל מטלה עם תאריך אחרון יוצאת כ-event בקובץ `deadlines.ics` (עם תזכורת יום מראש).
- פתיחה כפולה מייבאת ישירות ל-Google Calendar / Outlook / Apple Calendar.

### ייצוא ציונים (v1.8.0)
- טוגל בהגדרות (כבוי כברירת מחדל כדי שלא תשתף ציונים בטעות).
- אם מופעל — הקורס יוצא עם קובץ `ציונים.csv` מסונכרן.

### תפריט קליק-ימני (v1.8.0)
- 3 התנהגויות לבחירה בהגדרות:
  - **מיד** — מוריד את הקובץ מיד (ברירת מחדל).
  - **תור** — מוסיף לרשימה, אייקון התוסף מראה כמה בתור, כפתור "הורד את התור כ-ZIP" בפופאפ.
  - **שאל** — תפריט נפרד בכל קליק.
- Save As נפרד לקליק-ימני.
- "הוסף לתור" כאופציה תמידית בתפריט (גם כשהדיפולט הוא "מיד").

### הורדת מספר קורסים
- בעמוד "הקורסים שלי" — בורר עם צ'קבוקסים, חיפוש.
- כל קורס יורד כ-ZIP נפרד עם הדיפולטים החכמים.
- אופציה "רק חדשים" לעדכון מהיר.

### Zoom Cloud Recordings
- זיהוי דף `applications.zoom.us/lti/rich` שנפתח דרך LTI במודל.
- סריקה דרך מספר עמודים (Pagination אוטומטי).
- בורר הקלטות לפני פענוח URLs (להוציא הקלטות לא רלוונטיות).
- פענוח URL אמיתי לכל הקלטה — מעקב אחרי `window.open` של ה-React onClick → תופס את ה-`/rec/play/?tk=...` עם הטוקן שעובד מכל מכשיר.
- ייצוא לטקסט מסודר.

### דשבורד היסטוריה (v1.8.0)
- בדף ההגדרות: טבלה של כל הקורסים שהורדת, מתי, כמה פריטים, קישור לפתיחת הקורס מחדש.
- כפתור ניקוי היסטוריה (לא מוחק קבצים — רק את הזיכרון של ה-diff).

### עיצוב וחוויית משתמש
- לוגו ייעודי ב-4 גדלים, Dark mode אוטומטי, RTL מלא.
- Progress bar, חיפוש חי, ספירת נבחרים, לוג עם צבעים.
- נוטיפיקציה במערכת ההפעלה כשמסתיים.
- Badge מספר על אייקון התוסף לתור הקליק-ימני.

### תפקוד מתחת למכסה (לטובת מי שמתעניין)
- ZIP store-only שנכתב מאפס בטרנספורט, ללא תלות חיצונית.
- BOM אוטומטי בקבצי טקסט להצגה תקינה ב-Notepad.
- שמות קבצים מנוקים: תווים אסורים → `_`, נקודות בסוף שורה מוסרות (Windows איסור), סימני RTL מוסרים, רצף underscore מתמזג.

## שיפורים שמתוכננים לעתיד

### עדיפות גבוהה (תור עבודה הבא)

**חילוץ טקסט מהקלטות Zoom**  
ל-Zoom Cloud Recordings יש לעיתים VTT/SRT/transcript. נוסיף הורדה אוטומטית שלהם לתוך תיקיית ההקלטות, כדי שאפשר יהיה לחפש בתוך מה שנאמר בשיעור. צריך זהירות עם הקוד הקיים של ה-Zoom — לא לגעת בלי דיון.

**שילוב עם דף הדדליינים של מודל**  
במודל יש דף `/my/index.php` שמרכז את כל הדדליינים הקרובים. במקום (או בנוסף ל-) `deadlines.ics` הנוכחי, אפשר לשלוף משם כשנכנסים למודל ולהציג התראה כשמתקרבים דדליינים.

**שיפור Diff: זיהוי קבצים שעודכנו**  
היום diff מזהה "חדש" / "לא בדיפולט". להוסיף תווית **"עודכן"** לקבצים שקיימים אבל גודלם השתנה — כדי לא להוריד שוב את כל הקורס, אלא רק את מה שעודכן.

### עדיפות בינונית

**תצוגה מקדימה לקבצים**  
ב-popup, כשמרחפים על שורת קובץ, להציג גודל / סוג / מתי הועלה. שיפור חוויית בחירה לפני הורדה.

**שיפור AI**  
המשתמש מתכנן משהו בעתיד אבל לא כמו שהצעתי — מחכה.

### עדיפות נמוכה (נחמדים ל-have, לא קריטיים)

**הצפנת ZIP בסיסמה**  
דורש דחיסה אמיתית (כרגע אנחנו store-only). סיבוך טכני משמעותי לתועלת קטנה.

**גרירה לסידור קבצים**  
מאפשר למשתמש לסדר ידנית את סדר/קיבוץ הקבצים לפני הזיפ. תועלת מוגבלת.

**שמות תיקיות מותאמים אישית**  
תבניות כמו `{course-id}_{name}_{date}.zip` במקום השם הקיים. נחמד אבל לא חיוני.

**תמיכה במוסדות אחרים**  
אוניברסיטת תל-אביב, חיפה, BGU — כולן Moodle, אז אותה לוגיקה תעבוד. דורש בעיקר עבודת קונפיגורציה.

**תמיכה בשפות אחרות**  
ממשק באנגלית/רוסית/ערבית. כרגע עברית בלבד.

**כפתור צף בעמוד המודל**  
במקום לפתוח את הפופאפ, להזריק כפתור צף לעמוד הקורס עצמו. נחמד אבל לא קריטי.

### אופציות למפתחים (בהסבר פשוט)

**ייצוא JSON של הקורס** — קובץ `course.json` בתוך ה-ZIP שמכיל את כל המטא-דאטה (שם פריט, סוג, ID, תאריך, נתיב בזיפ). שימושי למי שרוצה לכתוב סקריפט שיעשה משהו עם הקורסים — למשל לבנות אינדקס חיפוש, לייצא ל-Notion, או לסנכרן עם כלי אחר.

**CLI / Headless** — חבילה שתעשה את אותה הורדה בלי דפדפן, מתוך שורת פקודה. שימושי לגיבוי תקופתי אוטומטי מהשרת, או למישהו שרוצה להוריד מאות קורסים בלי לפתוח Chrome.

**ספריית פלאגינים** — מאפשר למישהו להוסיף תמיכה בסוגי פעילויות חדשים (כמו H5P, SCORM אינטראקטיבי) בלי לגעת בליבת הקוד. דורש ארכיטקטורה מסודרת.
