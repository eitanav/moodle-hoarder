## 🟠 פיצ'רים חדשים — סדר עדיפות שהמשתמש ביקש

### דף הגדרות נפרד (תשתית — קודם)
- [ ] יצירת **דף הגדרות עצמאי** (`options.html`) שנפתח מהפופאפ או מקליק ימני על האייקון.
- [ ] כל הטוגלים הבאים יהיו שם, לא בפופאפ.
- [ ] שמירה ב-`chrome.storage.local`/`sync`.

### טוגלים בדף ההגדרות
- [ ] **מבנה ZIP**: רדיו של "תקיות סקשנים" / "Dump אחד שטוח" / "שניהם". ברירת מחדל = שניהם (כמו עכשיו).
- [ ] **Save As**: כשמופעל, פתיחת דיאלוג שמירה במקום ירידה אוטומטית ל-Downloads. ברירת מחדל = כבוי.
- [ ] **נתיב הורדה מותאם**: שדה טקסט לנתיב יעד. אם ריק → תיקיית Downloads. רלוונטי גם להורדת מספר קורסים.
- [ ] **כלילת ציונים**: כבוי כברירת מחדל (כדי שלא ישתפו ציונים בטעות).
- [ ] **סוגי קבצים להורדה**: צ'קבוקסים לפי `type` (`resource`, `folder`, `assign`, `url`, `page`, `book`, `forum`, וכו'). ברירת מחדל = הקיימת.
- [ ] **התנהגות קליק-ימני**: רדיו של "הורד מיד ל-ZIP חדש" / "הוסף לתור" / "שאל כל פעם".
- [ ] **Save As בקליק-ימני**: טוגל נפרד.

### חידוש הורדה כושלת (fail-safe)
- [ ] שמירת checkpoint לכל קובץ שירד בהצלחה ב-`chrome.storage.local` (key לפי `courseId`).
- [ ] אם הפופאפ נסגר/נופל באמצע — בסריקה הבאה לזהות checkpoint קיים ולשאול "להמשיך מאיפה שעצרת? (X/Y פריטים ירדו)".
- [ ] ניקוי ה-checkpoint רק כשההורדה הסתיימה (כולל אריזת ה-ZIP).

### שליפת ציונים
- [ ] סריקה של `grader/report.php?id=<courseId>` לכל קורס.
- [ ] ייצוא ל-CSV (UTF-8 עם BOM) בתוך ה-ZIP כ-`ציונים.csv`.
- [ ] רק אם הטוגל בהגדרות מופעל.

### קליק-ימני מתקדם
- [ ] תפריט מתפצל לפי הטוגל בהגדרות: "הורד מיד / הוסף לתור".
- [ ] אם "הוסף לתור" — אינדיקטור (badge על האייקון) שמראה כמה בתור.
- [ ] בפופאפ — כפתור "הורד את התור כ-ZIP" כשיש פריטים בתור.
- [ ] תמיכה ב-Save As לפי הטוגל.

### שיפורי "מצב Diff"
- [ ] קבצים עם אותו שם ושינוי בגודל → תווית **"עודכן"** (במקום "חדש").
- [ ] פריטים שאינם בברירת המחדל (פורומים, צ'אטים) שלא ירדו בסיבוב הקודם → תווית **"לא בדיפולט"** (במקום "חדש").
- [ ] השינויים האלה גם בקובץ `info.txt`.

### דשבורד היסטוריה (בדף ההגדרות, לא בפופאפ)
- [ ] טבלה של כל הקורסים שהורדת: שם, תאריך אחרון, כמה פריטים, גודל ZIP, כפתור "הורד שוב".
- [ ] כפתור "נקה היסטוריה" (לא מוחק קבצים — רק את הזיכרון של ה-diff).
